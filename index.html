What is CSS?

  1. CSS stands for Cascading Style Sheets.
  2. CSS describes how HTML elements are to be displayed on screen paper, or in other media.
  3. Css is not neccesary CSS saves a lot of work. I can controls the layout of multiple web pages all at once.
  4. External stylesheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways :

  • Inline-by using the style attribute inside HTML
  • Internal-by using style element in the head section
  • External-by using a link element to link an external CSS file.
  • The most common way to add CSS is to keep the styles in external css files. However, in this tutorial we will use inline and intrenal styles, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs id, the difference is that CSS class applies a style to multiple elements. ID, on the other hand, applies a style to one unique element. ID is also special in that you can use a secial URL to linkl directly to an element and it's used by Javascript.

In CSS, selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of styles can be applied to nthe elements.

There is a wide range of selectors available. Two of the most commonly used are class and ID. Both are used to target ele,ments to which a style should be applid.
